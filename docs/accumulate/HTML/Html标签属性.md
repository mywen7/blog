# HTML标签属性

比如
<font color="red">class，id，style</font>
这些都是html标签的属性，不过自己一时大意忘记了。  
<font color="red">title</font>
也是基本所有html标签都有的属性。  
<font color="red">alt</font>
是\<img>标签仅有的属性，  
<font color="red">href</font>
是链接属性，target属性是指定跳转目标  
<font color="red">name，type, value, maxlength，radio，checkbox，checked，reset，password，file</font>
属性在\<input>上存在（其他好多标签上也有name，type），等等，所以html标签上自身可以使用的标签挺多的。  
<font color="red">src</font>
引入属性啊，  
很久之前的
<font color="red">border，width，height，aling，cols</font>
（指定表格栏数），
<font color="red">rows</font>（指定表格列数），
<font color="red">clospan</font>（合并栏数），
<font color="red">rowspan</font>（合并列数），
<font color="red">frame，bgcolor，action，behavior，direction，loop，onmouseover，color，size</font>
等等大多可以用CSS来代替了（以至于让我都不记得html自身还有这么多属性，一直以为html就只剩标签了 ，连class，id，style也是他的属性都忘记了，，真是脑子不中用了。）
