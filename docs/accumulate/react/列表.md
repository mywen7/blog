但是，还是跟Vue一样需要加上Key值作为唯一标志符。  

如果列表项目的顺序可能会变化，我们不建议使用索引来用作 key 值，因为这样做会导致性能变差，还可能引起组件状态的问题。所以最好是使用id来作为key值
