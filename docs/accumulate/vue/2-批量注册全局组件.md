# 批量注册全局组件(2)  
![image](./assets/pi-1.png)  

<b>require.context是webpack中的方法，相当于export跟import。</b>  

使用方法：  
<font color="red">require.context( directory,  useSubdirectories,  regExp )</font>

1. directory: 要查找的文件路径
2. useSubdirectories: 是否查找子目录
3. regExp: 要匹配文件的正则
